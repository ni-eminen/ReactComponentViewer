 ## Installation
 1. Navigate to ReactComponentViewer folder
 2. Install dependencies:
 
 ```bash
 poetry install
 ```
 3. Run the program via Poetry:
 
 ```bash
 poetry run invoke start
 ```
 ## Command-line functionalities
